Description

Voici mon site web sur les interventions réalisées par les pompiers en France en 2021. J’ai sélectionné 5 interventions pertinentes que vous pourrez examiner pour une région précise.
Vous pourrez choisir l’intervention qui vous intéresse dans la rubrique située sur la barre de navigation. Vous y trouverez un tableau accompagné d’un graphique, organisés par département, montrant le nombre d’appels reçus.
